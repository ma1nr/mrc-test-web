Расписание Минского радиотехнического колледжа  
Автор: Кунин Ярослав, группа 4к9392  
Версия: 1.0

Веб‑приложение для просмотра расписания занятий колледжа.  
Проект поддерживает:

- расписание по группам  
- расписание по преподавателям  
- расписание по предметам  
- матрицу расписания (группы × пары)  
- половины пары (half1 / half2)  
- выбор даты (сегодня / завтра / кастом)  
- адаптивный интерфейс  
- анимации и современный UI  

Инструкции по запуску:

1. Копирование репозитория

git clone https://github.com/ma1nr/mrc-test-web.git
cd <mrc-test-web>

2. Создание виртуального окружения
python -m venv venv

3. Активация
Windows: venv\Scripts\activate
Linux/Mac: source venv/bin/activate

4. Применение миграций ДБ
python manage.py migrate

5. Создание суперпользователя
python manage.py createsuperuser

6. Запуск сервера
python manage.py runserver

Ссылка на сайт: http://127.0.0.1:8000/
Ссылка на админку: http://127.0.0.1:8000/admin/
